# Counting Games for a 2-Year-Old

A family guide for helping a toddler learn counting through short, playful activities.

***

--- PAGE BREAK ---

# 1. Put-in-the-Box Count

**Objective**
Practice counting objects one by one while placing them into a container.

**Prep**
3–5 blocks, balls, or soft items; 1 small box or basket.

**Instructions**

1. Place the items beside the box.
2. Ask the child to put one item inside at a time.
3. Count aloud together as each item goes in.
4. Repeat with different numbers of items.

***

# 2. Build-and-Count Tower

**Objective**
Learn counting by building a tower and adding one block at a time.

**Prep**
6–10 blocks or stacking cups.

**Instructions**

1. Give the child one block to start.
2. Add one block at a time.
3. Count each block aloud as the tower grows.
4. Ask, “How many now?” at the end.

# 3. Clip the Cards

**Objective**
Match the number of clips to the number of dots on a card.

**Prep**
Paper cards with 1–5 dots; clothespins or plastic clips.

**Instructions**

1. Show one card at a time.
2. Ask the child to count the dots.
3. Help the child clip the same number of clips onto the card.
4. Repeat with other cards.

***

# 4. Feed the Toy

**Objective**
Count by “feeding” a toy the correct number of items.

**Prep**
A toy animal, cup, or box; small counters, balls, or paper pieces.

**Instructions**

1. Tell the child that the toy is hungry.
2. Ask for a small number, such as 1, 2, or 3.
3. Count each item as it is given to the toy.
4. Praise the child for finishing the task.

***

# 5. Sort and Count

**Objective**
Group objects and count each group.

**Prep**
Mixed items such as blocks, balls, clips, or buttons.

**Instructions**

1. Place all the items together in one space.
2. Ask the child to group the same items together.
3. Count each group after sorting.
4. Repeat with new items or fewer pieces.

***

# 6. Dice and Do

**Objective**
Connect numbers with actions and movement.

**Prep**
1 die; enough open space to move safely.

**Instructions**

1. Roll the die together.
2. Say the number aloud.
3. Ask the child to do that many actions, such as claps, stomps, or jumps.
4. Take turns rolling and acting.

***

# 7. Match the Basket

**Objective**
Match equal quantities in separate containers.

**Prep**
2–4 bowls or baskets; matching counters, balls, or blocks.

**Instructions**

1. Put a small set of items in front of the child.
2. Ask the child to place the same number into each basket.
3. Count each set together.
4. Increase the quantity slowly over time.

***

# 8. Step Count Trail

**Objective**
Count steps while following a simple path.

**Prep**
Tape, cushions, or pillows to make a short path on the floor.

**Instructions**

1. Mark out a small path.
2. Ask the child to take 2 steps, then 3 steps, then 4 steps.
3. Count each step aloud together.
4. End with a small reward or target item.

***

# 9. Snack Counting

**Objective**
Count snacks as part of a daily routine.

**Prep**
Small snack portions such as crackers, raisins, or cereal pieces.

**Instructions**

1. Place the snack pieces on a plate.
2. Count them together before eating.
3. Ask the child to give one piece at a time.
4. Repeat during snack time as appropriate.

***

# 10. Jump and Count

**Objective**
Combine counting with gross motor movement.

**Prep**
Open floor space; optional floor markers.

**Instructions**

1. Ask the child to jump a small number of times.
2. Count each jump aloud.
3. Start with numbers such as 1, 2, or 3.
4. Let the child choose your turn as well.

***

# 11. Hidden Counters

**Objective**
Strengthen counting, memory, and curiosity.

**Prep**
A cup, sock, or opaque container; 2–4 small counters.

**Instructions**

1. Hide the counters inside the container.
2. Shake the container gently.
3. Let the child guess how many are inside.
4. Tip them out and count together.

***

--- PAGE BREAK ---

# 12. Make It Equal

**Objective**
Copy a set and match quantities.

**Prep**
Blocks, clips, or balls; a flat surface.

**Instructions**

1. Place a small number of items in front of the child.
2. Ask the child to make his set look the same.
3. Count together to check the match.
4. Repeat with another set.

# Family Use

These activities work best when they are short, playful, and repeated often. Family members can use the same wording each time so the child hears the number language consistently. Keep the games light and encouraging, and stop while the child is still interested.
